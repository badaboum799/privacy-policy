Politique de Confidentialité - Only AI

Dernière mise à jour : Décembre 2024

Only AI s'engage à protéger votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

1. Informations que nous collectons

  • Profil utilisateur : Prénom/pseudo, âge, genre, centres d'intérêt
  • Photos : Photos de profil (optionnelles)
  • Messages : Conversations avec les IA
  • Données d'utilisation : Interactions avec l'application

2. Utilisation des autorisations

  • Caméra : Uniquement pour prendre des photos de profil. Aucune photo n'est partagée sans votre consentement.
  • Stockage : Pour sauvegarder vos préférences et conversations.

3. Comment nous utilisons vos données

  • Personnaliser votre expérience avec les IA
  • Améliorer nos services
  • Assurer la sécurité de l'application
  • Gérer les paiements (si applicable)

4. Partage des données

Nous ne vendons JAMAIS vos données. Vos informations sont utilisées uniquement pour le fonctionnement de l'application.

5. Sécurité

Vos données sont protégées par :

  • Firebase (Google) - standards de sécurité élevés
  • Chiffrement des données sensibles
  • Accès restreint aux développeurs

6. Vos droits

  • Consulter vos données
  • Modifier vos informations
  • Supprimer votre compte et toutes vos données
  • Exporter vos données

7. Conservation des données

Vos données sont conservées tant que votre compte est actif. Elles sont supprimées définitivement lors de la suppression du compte.
